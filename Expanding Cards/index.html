Expanding Cards

Explanation: Click on a panel to expand it and see the title. Using the class 'active', I set up a JS script to detect when a panel is clicked. If it's clicked, 'active' is removed from all panels and only reassigned to the clicked panel. When a panel has the class 'active', the panel expands to show more of the image and a title fades in.
